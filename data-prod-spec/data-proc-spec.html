Data Product Specification

At a core, Data mesh is a graph, a network, consisting of nodes and connecting edges. Each node in a Data Mesh is called Data Product. According to J.Majchrzak at al., Data Product is an autonomous, read-optimized, standardized data unit containing at least one dataset (Domain Dataset), created for satisfying user needs.
